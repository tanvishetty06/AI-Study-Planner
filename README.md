📚 AI Study Planner

An AI-powered study planner built using Python, Streamlit, Pandas, and Plotly.

🚀 Features

- 🔐 Login Page
- 📚 Study Planning Dashboard
- 📈 Progress Tracker
- 📊 Interactive Charts
- 🤖 AI Study Suggestions
- 🌟 Daily Motivation Quotes
- 🎨 Beautiful User Interface

🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

▶️ Run the Project

pip install -r requirements.txt
streamlit run main.py

💡 Purpose

This project helps students organize their studies, track progress, and stay motivated using an interactive AI-style planner.
